# Segundo Examen Parcial - I Parte

Proyecto desarrollado para la primera parte del Segundo Examen Parcial del curso **Diseño y Programación de Plataformas Móviles (EIF411)**.  
I Ciclo - 2026.

## Descripción
Aplicación de ejemplo en Android para demostrar el uso básico de fragmentos (`Fragment`). Permite listar un conjunto de países y alternar su visualización entre formato de lista y grilla de dos columnas utilizando un Spinner.
